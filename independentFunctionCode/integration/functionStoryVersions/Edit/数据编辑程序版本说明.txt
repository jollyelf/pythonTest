dataEditV1.0：
	初始的程序功能，包括：
	1、删除重复的几何要素
	2、填写每一个要素的管段编码，为所有的设备设施要素找寻其所归属的管段，并将管段编码填入要素表中
		三通表需要重新检查
	3、填写每一个要素的坐标，对所有的设备设施要素进行其坐标计算
	4、对每一个设备设施要素进行编码，利用编码规则对设备设施给定一个初始编码，
		对于不能明确类型的设备设施，需要后期修改特征编码字母
	5、将设备设施的一些特殊字段利用管段信息来填写，利用管段的信息来填写编码、管线名称、
		管段名称、投用日期、设计单位、施工单位、监理单位、检测单位、竣工图纸名称及编号
		采集日期、采集单位、录入时间
dataEditV2.0：
	在V1.0的基础上，新增功能，包括：
	1、新增管线表填写功能（包括其几何属性和非几何属性）。将管段数据汇总，填入管线表中
	2、新增穿跨越填充功能。将敷设方式为穿跨越的管段写入穿跨越表中
	3、待穿跨越表填写完成后，将有套管的穿跨越表写入套管表中
dataEditV3.0：
	在V2.0基础上，新增功能，包括
	1、新增设备设施的里程填写功能
	2、新增数据库要素的重新编码功能（此功能在对要素类型进行确定，并修改其特征编码后使用）
	3、修订对设备设施特征参数的填写，将三通主管管径单独提出来填写
	4、新增封堵物几何信息填写功能
dataEditV3.1：
	在V3.0基础上，完善了部分功能，包括：
	1、完善了穿跨越编辑功能；
	2、完善了三通编辑功能；
	3、新增默认值填写功能；
	4、完善了设备设施特殊字段编辑功能。
dataEditV3.2：
	在V3.1基础上，依据20190419讨论内容修改，编辑程序，包括
	1、穿越道路默认三级及以下
	2、阀门功能默认为截断带放散
	3、设备的连接方式默认值修改为，如果管材为PE管，其连接方式为热熔，如果为钢	     管则为焊接；
	4、弯头防腐材料可能与管段不一致，直接取消其默认值
